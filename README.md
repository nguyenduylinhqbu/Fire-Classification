# Fire-Classification
## Installation

* Install requirements in your environment. 

* Clone this repository.

* Download fire datasets by following the instructions below and out into "Dataset" folder.
## Datasets
* We provide 6 datasets
  * FireNet
  * FireSense
  * CairFire
  * FireSmoke
  * FireDetection
  * ThermalFire (Proposed dataset)
